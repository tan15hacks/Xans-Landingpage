# Xans Landing Page

A premium responsive landing page concept for **Xans**, a minimal aesthetic clothing apparel brand.

This project is part of a landing page portfolio focused on modern front-end development, clean UI, responsive layouts, and conversion-focused sections.

## Built with

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React icons

## Sections

- Sticky navigation
- Editorial hero section
- Moving brand strip
- Lookbook preview
- Product cards
- Collection feature section
- Email capture CTA
- Footer

## Local setup

```bash
npm install
npm run dev
```

Then open:

```bash
http://localhost:3000
```
